Выбор индекса для города

== Description ==

При помощи этого плагина можно добавить на страницу, пост или другую сущность, меню для выбора индекса по штату и городу. 

== Installation ==

1. Загрузите плагин в папку '/wp-content/plugins/'.
2. В разделе 'Плагины' активируйте плагин.
3. В разделе 'Настройки' зайдите в 'Zip-codes' и выберите для какой сущности будет применяться плагин и нажмите 'Save Changes'.
4. Перейдите в выбранную сущность и внизу появится новый пункт 'Your zip-codes'.
